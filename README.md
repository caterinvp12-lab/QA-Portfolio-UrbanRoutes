# QA Testing Project: Urban Routes
Este proyecto forma parte de mi formación como **QA Engineer**. En él, realicé el aseguramiento de calidad del módulo de tarifas y tiempos de la aplicación Urban Routes.

## 🛠️ Herramientas utilizadas
* **Google Sheets / Excel:** Diseño de casos de prueba atómicos.
* **Análisis de Requisitos:** Identificación de particiones de equivalencia para franjas horarias.

## 📋 Contenido del Repositorio
* **Casos de Prueba:** Validación de costos de viaje y tiempos de conducción en 6 franjas horarias críticas.
* **Criterios de Aceptación:** Verificación de la lógica de negocio del sistema.

Ingeniera Industrial en transición a QA Engineering, enfocada en la optimización de procesos y la integridad de datos.
